Versión de Unity utilizada: Unity 2021.2.16f1

Para tener en cuenta:
El juego se encuentra en la ruta -> Assets/Scenes/Pong

En el juego, al pasar a la fase donde se juega Pong se usan los siguientes controles:

- Flecha arriba y flecha hacia abajo.
- W y S.

Pueden usar cualquier combinación de estas teclas para mover la
plataforma que controlan hacia arriba o hacia abajo, respectivamente.

La plataforma izquierda es la que se puede controlar, la derecha pertenece
a la inteligencia artificial.

Cuando algun jugador gane, se puede reiniciar el juego oprimiendo la 
tecla R.
